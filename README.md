# GDGoC-25-02-Toyproject
[25-26 GDG on Campus Sookmyung 6기] 1분기 스터디(Spring 입문) 후 토이 프로젝트를 위한 저장소

## 팀명 : SS (더블 에스)
## Member : 빈서윤, 현소윤

## 선택한 구현 기능
### 1. 좋아요 기능
### 2. 이미지를 포함한 게시글 작성 및 조회 기능

## 깃 규칙
### Type: Subject
### (선택) Body
### (선택) Footer

Type : 어느 종류의 작업인지

Subject: 커밋 내용 요약 (ex: ~~~기능 추가)

Body: 추가적으로 긴 설명이 필요한 경우

Footer: 특정 이슈를 닫을 때 사용

## 브랜치 규칙
#### 1. 각자 브랜치를 생성해서 개인 브랜치에서 작업
#### 2. 브랜치 이름은 알아보기 쉽게 작성
#### 3. 브랜치에서 작업 완료 후 PR 남기면 상대방이 확인하고 code review 달아주기
#### 4. review에 이슈사항이 없을 시 main에 Merge
#### 5. 서로 PR을 남긴 후 연락을 남겨 빠르게 소통 하기!
